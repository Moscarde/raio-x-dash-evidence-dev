# Raio-X Municipal

Camada analítica do ecossistema **Raio-X Municipal**: dashboards e páginas de leitura sobre saúde pública municipal, construídos a partir do banco alimentado pelo projeto de engenharia de dados (`raio-x-engenharia`).

Comece por aqui:

- **[Status das fontes](/status-fontes)** — o que está carregado, para qual período e para quantos municípios.
- **[Perfil municipal](/perfil-municipal)** — visão geral de um município (hoje: Rio de Janeiro, RJ).
- **[Internações](/internacoes)** — produção hospitalar (SIH) por competência.

## Sobre o estágio atual dos dados

Este produto está em construção. Hoje as fontes de fato (SIH, SIM, SINASC, FNS, SIOPS, SISAB) cobrem **um único município — Rio de Janeiro (RJ)** — e a base de produção ambulatorial (SIA) ainda não está pronta no projeto de engenharia. Páginas de comparação entre municípios ficam para quando houver mais de um município com dado real carregado. Detalhes em [Status das fontes](/status-fontes).
